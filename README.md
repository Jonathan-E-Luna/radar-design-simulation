# radar-design-simulation
MATLAB simulation for an airborne 150 gHz surveillance radar. Computes geometry, SNR chain (T/N, C/N, T/(C+N)), swath width, Doppler precision, and probability of detection across near, boresight, and far range. 
Built for EEL-557 Intro to Radar Systems, UCF Spring 2025.

Achieved 100% mandatory spec compliance (16/16) and places 3rd in the course design competition, outperforming graduates teams as a solo competitor.
